# Governance

## Project Ownership

This project is owned and maintained by **ShadowAISolutions**.

## Decision Making

All decisions regarding the project direction, feature additions, and architectural changes are made by the project maintainers.

## Roles

| Role | Responsibilities |
|------|-----------------|
| **Owner** | Final authority on project direction, merges, and deployments |
| **Maintainer** | Code review, issue triage, and day-to-day project management |
| **Contributor** | Submits changes via pull requests following the [Contributing](CONTRIBUTING.md) guidelines |

## Releases

- Deployments are automated via GitHub Actions
- Pushes to `claude/*` branches are auto-merged and deployed
- Version numbers are tracked in code files (`VERSION` in `.gs`, `build-version` in HTML)

## Changes to Governance

This governance model may be updated as the project evolves. Changes will be documented in the [CHANGELOG](CHANGELOG.md).
